About CSS

  • Control various visual aspects such as colors, spacing,typography,borders and layout
  • Accept different types of values, including keywords,numerical values with units and functions
  • Many related properties can be combined into a single shorthand property for conciseness
  • Modern CSS includes logical properties that define layout based on writing modes

Inline CSS

Inline CSS refers to the method of applying Cascading Style Sheet (CSS) rules directly to individual HTML elements using the style attribute within the element's opening tag. 

Eg: <h4 style="color:black;
                padding-left:15px;
                font-size: larger;" > Inline CSS </h4 >

Internal CSS

Internal CSS, also known as embedded CSS, is a method of applying styles directly within a single HTML document. It involves placing CSS rules within a <style> element.

Eg: <style>
        .img{
            padding-left:30px;
            padding-right: 10px;
        }
</style>

External CSS

External CSS refers to a method of applying styles to HTML documents by linking a separate CSS file to the HTML. This approach separates the presentation (CSS) from the content (HTML)

                h1{
    color:brown;
    background-color: bisque;
}
li{
    color:brown;
}


<link href="external.css" rel="stylesheet">
